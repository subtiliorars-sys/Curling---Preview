# Curling — Preview

Public **demo / preview** build for [PixelSports Curling](https://github.com/subtiliorars-sys/Curling) — a pixel-art curling **card roguelite**: play shots, spend handle, survive the bracket on the sheet.

## What this repo is

This repository holds a **sanitized, static web snapshot** (GitHub Pages) of a playable slice — not the full private product codebase. Open `index.html` or the published Pages URL to try the House Tournament curling-cards experience.

**Not included here:** source TypeScript, design docs, build pipeline, or unreleased features from the HQ repo.
